BB8 Controller
==============
BB8 Controller built using [Electron](https://electron.atom.io) with TouchBar interactions.

How to run?
-----------
```bash
# You need to have a Mac with TouchBar support for this
npm install
npm start # dev instance
npm package # package Mac app
```
